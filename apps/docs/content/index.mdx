# Welcome to Better i18n

Better i18n is a **GitHub-first localization platform** for engineering teams. It automatically discovers translation keys from your repository, keeps translations synchronized, and ships changes back as pull requests or direct commits.

## Core Features

- **Automatic Key Discovery**: Scans your codebase to find translation keys
- **AI-Powered Translation**: Suggests translations with human approval workflow
- **GitHub Integration**: Pull request workflows for translation updates
- **CDN Delivery**: Global edge network for fast translation loading
- **Developer Tools**: CLI, MCP server, and framework SDKs
